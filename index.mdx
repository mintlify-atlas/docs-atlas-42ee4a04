<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4 text-center">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8 text-center max-w-2xl mx-auto">
    Get up and running with Grok Search MCP Server in minutes
  </p>

  <Steps>
    <Step title="Get your xAI API key">
      Visit the [xAI Developer Portal](https://console.x.ai/) to create an account and generate your API key.
      
      ```bash
      # Your API key will start with "xai-"
      export XAI_API_KEY="xai-your-api-key-here"
      ```
    </Step>
    
    <Step title="Install the MCP server">
      Clone and install the server globally using npm:
      
      <CodeGroup>
      ```bash npm
      git clone https://github.com/stat-guy/grok-search-mcp.git
      cd grok-search-mcp
      npm install -g .
      ```
      
      ```bash npx
      # Or use directly with npx (no installation needed)
      npx grok-search-mcp
      ```
      </CodeGroup>
    </Step>
    
    <Step title="Configure Claude Desktop">
      Add the server to your Claude Desktop configuration file:
      
      **macOS**: `~/Library/Application Support/Claude/claude_desktop_config.json`
      
      **Windows**: `%APPDATA%/Claude/claude_desktop_config.json`
      
      ```json
      {
        "mcpServers": {
          "grok-search": {
            "command": "npx",
            "args": ["grok-search-mcp"],
            "env": {
              "XAI_API_KEY": "your-api-key-here"
            }
          }
        }
      }
      ```
    </Step>
    
    <Step title="Restart Claude Desktop">
      Completely quit and restart Claude Desktop. The Grok Search tools will now be available for use.
      
      <Tip>Use the `health_check` tool to verify your setup is working correctly.</Tip>
    </Step>
  </Steps>
</div>
